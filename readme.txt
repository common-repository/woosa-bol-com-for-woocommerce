== Description ==

Wil je meer omzet draaien met je webshop? Dan is de bol.com WooCommerce plugin voor WordPress onmisbaar! bol.com is de winkel van ons allemaal. Bol.com heeft een breed product assortiment. Het is dus voor iedere webshop interessant om zijn producten te verkopen via de website van bol.com.

De bol.com WooCommerce plugin - light version, stelt je in staat om je producten eenvoudig te verkopen via de website van bol.com. De website heeft meer dan 1 miljoen unieke bezoekers per maand. Een effectieve manier van marketing met goede resultaten. Daarnaast ondersteunt bol.com in de verzending door het beschikbaar stellen van pakbonnen, maar ondersteunt zij ook in de retourprocedure. Wel zo makkelijk!

Belangrijke functionaliteiten van de bol.com WooCommerce plugin - light version
*	Publiceer max. 5 producten vanuit WooCommerce op bol.com
*	Voeg de EAN code handmatig toe aan de bol.com instellingen
*	Stel per handmatig product de vereisten instellingen in zoals prijs, verzendkosten, voorraad etc.

Belangrijke functionaliteiten van de bol.com WooCommerce plugin - pro version
* 	Herken EAN code automatisch via de SKU of een Custom Field
*	Bulk edit / publicatie van bol.com producten
*	Plaatsen van jouw producten naar keuze op bol.com
*	Automatisch toevoegen van bestellingen aan je WooCommerce webshop vanuit bol.com
*	Je kunt je eigen verkoopprijs en verzendkosten bepalen
*	Gebruik de WooCommerce prijs + een marge bedrag of percentage
*	Automatische retourafhandeling via bol.com
*	2-way synchronisatie tussen WooCommerce en bol.com
*	Meerdere bol.com accounts in een WooCommerce webshop

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woosa-bol-com-for-woocommerce` directory, or install the plugin through the WordPress plugins section directly.
1. Activate the plugin through the 'Plugins' section in WordPress
1. Go to bol.com->Settings to configure the plugin

== Frequently Asked Questions ==

= Hoe kan ik mijn producten publiceren op bol.com? =

Elk product in WooCommerce dient minimaal een EAN code te hebben. Op basis daarvan kan het product automatisch worden toegevoegd aan bol.com.

= Mijn EAN code is onbekend =

Als jouw EAN code onbekend is bij bol.com dien je alle product informatie zelf aan te leveren. Je kunt ook een FTPS account bij bol.com aanvragen om dit proces te automatiseren middels de plugin.

= Waarom verkopen via bol.com? =

Bol.com heeft miljoenen bezoekers per maand. Jij kunt mee profiteren van dat bezoekersaantal door je producten te publiceren op hun website.
